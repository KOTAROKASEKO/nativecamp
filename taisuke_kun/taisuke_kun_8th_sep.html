Case File #01: 薬局のアプリ

Scenario Analysis

原文:
太郎のお母さんは薬局のアプリを持っていました。店員にQRをスキャンして、１時間後、彼の家で、お母さんはお金がポイントになることを太郎に説明しました。数日後太郎は、ポイントがもらえると思って、アプリで買い物をしました。

Case File #04: ポイントの説明

Expression Check

[ Question 3 ]

「お母さんはお金がポイントになることを太郎に説明しました。」の翻訳比較

たいすけくんの回答:

"Taro's mom explained that money become point."

模範回答:

"We can get points from the purchase."

模範回答のように「購入によってポイントがもらえる」と表現する方が自然です。たいすけくんの回答をより良くするために、動詞 "become" の代わりに使える、より適切な表現を考えてみよう。

Case File #05: 最後の挑戦

Final Translation

[ Question 4 ]

「数日後太郎は、ポイントがもらえると思って、アプリで買い物をしました」の翻訳比較

たいすけくんの回答:

"Few days later, Taro thought that Taro can get point, so he shopping by application."

模範回答:

"A few days later, Taro thought that he could get points, so he used the app when he was paying to gain points."

模範回答はより具体的で自然な表現を使っています。特に「アプリで買い物しました」の部分は、どのような状況を想像して翻訳されているか、考えて説明しなさい。

Mission Report / 解答と解説

Case File #04: ポイントの説明

解説:
「お金がポイントになる」を直訳すると不自然になりがちです。"Money become points" よりも、「（買い物などをすることで）ポイントを稼ぐ、得る」という発想で、"earn points" や "get points" という動詞を使うのが一般的です。例えば、"You can earn points with every purchase." のように表現できます。

Case File #05: 最後の挑戦

解説:
「アプリで買い物」は、具体的に「支払いの時にアプリを使った」と解釈すると、より英語らしい表現になります。模範回答の "he used the app when he was paying" は、まさにその状況を表しています。単に "shopping by application" と言うよりも、行動が具体的で分かりやすくなります。
