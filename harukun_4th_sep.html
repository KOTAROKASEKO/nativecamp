もんだい① 「パーティーに参加する」をえいごで言うと？

こたえ → join the party

もんだい② 「私はパーティーに参加したいです」をえいごで書いてみよう。

こたえ → I want to join the party.

もんだい③ 「彼はパーティーに参加したいですか？」をえいごで書いてみよう。

こたえ → Does he want to join the party?

もんだい④ 「いいえ、彼はパーティーに参加したくありません。」をえいごで書いてみよう。

こたえ → No, he doesn't want to join the party.

もんだい⑤ 「do」と「does」のつかいわけ。
（　）にはいるのはどっちかな？

1. あなた、わたし、かれら など -> (      ) をつかう。

2. かれ、かのじょ、ジョン など -> (      ) をつかう。

こたえ → 1. do / 2. does

もんだい⑥ 「彼はバイクが好きではありません。」をえいごで書いてみよう。

こたえ → He doesn't like bike.

もんだい⑦ 「彼女は武道をやります。」をえいごで書いてみよう。

こたえ → She does Budo.
